# Adhesiones 2026

Gestor de Afiliaciones Políticas optimizado para trabajo de oficina y carga rápida.

## Características
- Registro de afiliados con validación de DNI.
- Carga de fotos de DNI (Frente y Dorso) con soporte para pegar desde el portapapeles.
- Compresión automática de imágenes.
- Buscador por DNI y Nombre.
- Vista de impresión optimizada para A4 (Ficha de Adhesión).
- Autenticación segura con Supabase.

## Tecnologías
- Next.js 15+ (App Router)
- Tailwind CSS
- Supabase (Database, Auth, Storage)
- Lucide React (Icons)
- Browser Image Compression

## Configuración
1. Clonar el repositorio.
2. Crear un archivo `.env.local` con las credenciales de Supabase.
3. Ejecutar `npm install`.
4. Ejecutar `npm run dev`.
